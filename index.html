What is this?

An Answer Set Programming system for generating overworld maps for use with the Solarus engine.

Does it work?

Yes, albeit not very fast - the system

How do I use it?

First, set up the Quest Editor from the Solarus Project (this system is tested on version 1.5.3).

Then, make sure that you have clingo 5 installed (from here), and that it's able to run. Additionally, you need either Python 2 or 3 in order to run the script.

With that in place, download the contents of the repo from here. Place the tool in the main quest folder for the Solarus Quest you which to generate maps for.

When all dependencies are installed, navigate to the folder in your favorite command line tool and issue the following commnad:
MapGen.py [MapCount] [Width] [Height] [paramFile] [outPutJSON]

Where the first 3 are integers, the paramFile is a customizable set of rules for controling the output. outPutJSON is any valid filename for a JSON file.

The maps will automatically be placed in the maps folder for your quest, ready to be included!
